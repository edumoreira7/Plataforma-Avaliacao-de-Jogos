# Plataforma-Avaliacao-de-Jogos
GameRate é um projeto acadêmico desenvolvido para a disciplina de Tópicos Avançados de Bancos de Dados. Trata-se de uma plataforma web de avaliação de jogos digitais, inspirada em IMDb, Letterboxd e Metacritic, onde usuários podem atribuir notas e compartilhar opiniões, aplicando na prática conceitos de modelagem e organização de dados.
